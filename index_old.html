Некоторые названия у американских заводчиков

 



 

Пользователь monarc23 на форуме http://www.backyardchickens.com/ предлагает следующие обозначения генов.

N = Нормальный дикого типа

S = Silver - рецессивный разбавленных

C = Корица - рецессивный, вызывает сокращение черный пигмента

W = белый - рецессивный. Возможность маскировки всех цветов

Sh = Splash - белый основной цвет с цветными пятнами.

B = Blue Faced - Доминирующие - единственный фактор (B), Двухместный фактор (BB)

P = Golden Pearl - Доминирующий

R = Красногрудый - рецессивный

Pd = Pied - цветные птицы с белыми маховыми перьями.

Ca = Карамель - В настоящее время определены только в Австралии <- никогда не видел этого!

Pe = розовый Eyed - еще в разработке, крайне редкий

I = Слоновая кость - корица + Silver комбинацию <- только слышал, никогда не видел

Sl = Slate - Silver + Blue

Sm = Smoky - корица + Silver (слоновая кость) + синий

Dv = Дарт Вейдер

Cu = Curly
